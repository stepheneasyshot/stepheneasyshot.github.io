# Compose的绘制机制

Compose Runtime 采用了一种特殊的数据结构，称为 Slot Table。

Slot Table 与常用于文本编辑器的另一数据结构 Gap Buffer 相似，这是一个在连续空间中存储数据的类型，底层采用数组实现。区别于数组常用方式的是，它的剩余空间，称为 Gap，可根据需要移动到 Slot Table 中的任一区域，这让它在数据插入与删除时更高效。
